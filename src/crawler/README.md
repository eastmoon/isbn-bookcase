# 爬蟲系統

此項目是基於 Python 來設計一爬蟲系統，並藉此蒐集 ISBN 編號在各書籍搜尋網站上的資料；而由於爬蟲擷取的內容存在複數資料的可能，需再透過相近資料比對，給予每筆資料不同的權重來判斷其可用價值。

### 操作方式

+ 測試執行

```
python test/[filename].py
```
